# Model Vetting Studio

This Capability Arena app renders the current isolated model-vetting report as a local-only capture surface before scene placement. It is governed by [MADR 0021](../../../docs/madr/0021-local-first-no-cloud-implementation-spikes.md), [MADR 0027](../../../docs/madr/0027-quest3-usb-webxr-smoke-gate.md), and [MADR 0028](../../../docs/madr/0028-iwsdk-sidecar-spike.md).

Decision posture: local review metadata only. This studio can show which fixed-camera screenshots and video captures are missing for Anny-compatible candidates, but it is not evidence for real-Anny output, B+ realism, scene readiness, Quest readiness, production readiness, learner readiness, clinical validity, or scoring validity.
