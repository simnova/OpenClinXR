# Model Vetting Arena Package

This package defines the local-only report contract for isolated humanoid model vetting before scene placement. It is governed by [MADR 0021](../../../../docs/madr/0021-local-first-no-cloud-implementation-spikes.md), [MADR 0027](../../../../docs/madr/0027-quest3-usb-webxr-smoke-gate.md), and [MADR 0028](../../../../docs/madr/0028-iwsdk-sidecar-spike.md).

Decision posture: Capability Arena package. It can score and validate review-safe model lab reports, but it is not a production runtime package and cannot support learner, Quest, production, clinical, or scoring readiness claims.

Promotion requires a successor MADR or architecture-rule update plus isolated model evidence, scene-placement evidence, provenance/license review, and runtime/browser proof.
