# Multi-Actor State Spike

**Status:** Superseded for production-shaped contracts by `@openclinxr/session-state`.

**Governing decisions:** [MADR 0014](../../../../docs/madr/0014-cellixjs-inspired-domain-contexts.md) and [MADR 0018](../../../../docs/madr/0018-first-class-communication-style-layer.md).

This package remains as architecture evidence for:

- Colyseus, `@colyseus/schema`, bitECS, and custom-state option evaluation.
- Phase 2 persistence-boundary experiments.
- Historical spike tests and documentation.

Production packages should import `@openclinxr/session-state` instead. This spike package is not evidence for realtime synchronization, production persistence, Quest spatial synchronization, LLM actor quality, or clinical assessment validity.
