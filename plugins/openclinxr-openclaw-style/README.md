# OpenClinXR OpenClaw-Style Plugin

This local Codex plugin packages the repo-native OpenClaw-style / OpenClaw-inspired workflow bridge for OpenClinXR.

It is not an external OpenClaw runtime, daemon, or product integration. It bundles a skill that points Codex back to the canonical repo files, commands, hooks, role memories, and guardrails.

Use the repo marketplace at `.agents/plugins/marketplace.json` to install it in Codex.
